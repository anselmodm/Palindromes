Palindromes

A palindrome is a word or phrase that reads the same whether you start from the beginning or end.

In our case, the method is not sensitive to differences between uppercase and lowercase letters 
and will only take into account the ASCII characters located between “A” and “Z” (or “a” and “z”). 
Spaces and special characters will be ignored.

Examples of palindromes:

▪ “Radar” (ignore differences between uppercase and lowercase letters)
▪ “O galo ama o lago” (ignore spaces, accented characters and exclamation marks)

In this practical activity, you should implement the boolean function ePalindromo(String) 
which, given a String passed as a parameter, returns a boolean value indicating whether that 
String is a palindrome or not.
